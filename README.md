# Pharmacy Management System

![Technology](https://img.shields.io/badge/Built%20with-Java-blue)

## 📋 Project Description

The **Pharmacy Management System** is a desktop-based application developed using Java Swing. It is designed to facilitate the management of a pharmacy by automating inventory tracking, order processing, and transaction history. The system provides a user-friendly graphical interface for pharmacists to manage drugs, process sales, and maintain an efficient workflow.

## 🎯 Features

- **Inventory Management:** Add, update, remove, and view drugs in the inventory.
- **Sales Processing:** Place orders and update transaction history.
- **Capacity Management:** Set the maximum capacity of the pharmacy.
- **Transaction History:** Maintain records of all transactions.
- **User-Friendly Interface:** Developed using Java Swing for an intuitive experience.

## 🛠️ Tech Stack

- **Programming Language:** Java
- **GUI Framework:** Java Swing
- **Development Kit:** Java Development Kit (JDK)

## 🚀 Getting Started

### Prerequisites
- Java Development Kit (JDK) installed (version 8+ recommended).
- A Java-compatible IDE (e.g., IntelliJ IDEA, Eclipse, NetBeans) or command-line tools.

### Installation & Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/your-repository/pharmacy-management-system.git
   cd pharmacy-management-system
   ```

2. **Compile the Source Code:**
   ```bash
   javac *.java
   ```

3. **Run the Application:**
   ```bash
   java Main
   ```

## 📌 Main Components

- **`Main.java`**: Entry point of the application, initializing the system.
- **`Pharmacy.java`**: Manages drug inventory, sales, and transactions.
- **`Drug.java`**: Represents a drug entity with attributes such as name, ID, price, and quantity.
- **GUI Classes:**
  - `WelcomeFrame.java`: Displays the welcome screen.
  - `MainFrame.java`: Provides access to different system functionalities.
  - `AddDrugFrame.java`: Allows pharmacists to add new drugs.
  - `RemoveDrugFrame.java`: Enables drug removal from inventory.
  - `UpdateDrugFrame.java`: Updates drug details.
  - `ViewInventoryFrame.java`: Displays available drugs.
  - `PlaceOrderFrame.java`: Handles order placement.
  - `TransactionHistoryFrame.java`: Shows past transactions.
  - `TotalSalesFrame.java`: Displays total pharmacy sales.
  - `CapacityFrame.java`: Sets the pharmacy's capacity.
